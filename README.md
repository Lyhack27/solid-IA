This is the official website for Solid IA, a company specializing in AI solutions, web development, cybersecurity, and automation services.

## Features

- Modern and responsive design
- Animated particle background
- Services showcase
- Contact form
- Careers section

- JavaScript
- Particles.js
- Font Awesome icons
